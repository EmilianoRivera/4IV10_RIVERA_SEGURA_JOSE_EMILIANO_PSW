VALIDACION DE FORMULARIOS

Existen dos métodos principales para la ontención de los datos dentro de un formulario web, para lo cual aquí lo explicaremos:

GET:

Este método es el stándar cómo salida del navegador mediante el cual se resuelve una petición por parte del cliente, así mismo en la salida estándar(url), se puede visualizar los datos enviados mediante el formulario.

POST:

Método es el más utilizazdo para el envío de información desde un formulario atendiedo una petición por parte del cliente, actuando para poder agregar o insertar nuevos datos en una DB,así mismo a diferencia del método GET, este utiliza un tipo de seguridad, mediante el cual la información no es visible por medio de la salida estándar del navegador.
